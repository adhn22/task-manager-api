# Task Manager API

REST API for personal task management built with Django REST Framework.

## Description

API that allows registered users to create, organize, and track
their personal tasks, with support for priorities, due dates, and statuses.

## Features (v1)

- User registration and authentication (JWT)
- Full task CRUD (create, read, update, delete)
- Filter tasks by status and priority
- Auto-generated API documentation (Swagger)

## Out of scope (v1)

- Shared tasks between users
- Email notifications
- Admin roles

## Tech Stack

- Python 3.12
- Django 5.x
- Django REST Framework
- PostgreSQL
- SimpleJWT
